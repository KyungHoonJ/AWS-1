# 팀 프로젝트 기술서

## 서버 구조

### BackEnd

- Express 서버
- API 서버

### FrontEnd

- Nginx 서버
- 웹 페이지

## 계정

### 사용자

#### 회원 가입

- 유저 정보 암호화 필요

#### 로그인

- 일정 시간 후 자동 로그아웃

#### 로그아웃

#### 회원 탈퇴

- 탈퇴 시 해당 정보 남김

### 관리자

#### 상위 관리자

- 하위 관리자의 작업 가능
- 하위 관리자 등록 가능
- 하위 관리자 삭제 가능

#### 하위 관리자

- 카테고리, 게시판 등록
- 카테고리, 게시판 수정
- 카테고리, 게시판 삭제
- 게시글 삭제 가능
- 회원 탈퇴 가능

## 게시글(채팅)

### 게시글(채팅) 등록

### 게시글 수정

### 게시글(채팅) 삭제

- 채팅 삭제 시 기존 채팅 기록 남기기
  - ex) 삭제된 채팅입니다.
- 게시글 삭제 시 하위 댓글 모두 삭제

## 댓글

### 댓글(채팅) 등록

### 댓글 수정

### 댓글(채팅) 삭제

  - ex) 삭제된 댓글입니다.
